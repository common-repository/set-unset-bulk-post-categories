Allows user to set unset the categories of all the posts in a bulk without editing the posts itself.

== Description ==

You can set unset the categories of the posts in a single screen & filter the posts here by author, category and date range to set/unset categories for post and finally you can edit / delete the post from same screen.

It will save your valuable Time by providing you easy to use use single page screen to set/unset categories for post at the same time you can edit/delete post from same screen too.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'set-unset-bulk-categories'
3. Activate Set Unset Bulk Post Categories plugin from your Plugins page.
4. A menu will be added at the admin panel called 'Set Unset Bulk Post Categories'. Click on it & you will see the desired plugin result.

= From WordPress.org =

1. Download set-unset-bulk-post-categories.
2. Upload the 'set-bulk-post-categories' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Set Unset Bulk Post Categories from your Plugins page.
4. A menu will be added at the admin panel called 'Set Unset Bulk Post Categories'. Click on it & you will see the desired plugin result.

== Frequently Asked Questions ==

= How does it work? =

After activation, a menu 'Set Unset Bulk Post Categories' is added on the Admin panel. When clicked, shows all the posts with its title as well as its categories on the left. You can set & unset the categories of corresponding posts according to  requirement. Hit the 'Submit' button and you are done.
